# Mô hình OSI

Mô hình **OSI (Open Systems Interconnection)** là mô hình tham chiếu kết nối các hệ thống mở.

## 7 Tầng của OSI

1. **Physical Layer** (Tầng Vật lý)
2. **Data Link Layer** (Tầng Liên kết dữ liệu)
3. **Network Layer** (Tầng Mạng) - *IP, Router hoạt động ở đây*
4. **Transport Layer** (Tầng Giao vận) - *TCP, UDP*
5. **Session Layer** (Tầng Phiên)
6. **Presentation Layer** (Tầng Trình diễn)
7. **Application Layer** (Tầng Ứng dụng) - *HTTP, FTP, DNS*
